Site web interactif : Pixel War

Ce projet réalisé en première année consistait à reproduire le fonctionnement de r/place sur Reddit. Si vous ne connaissez pas, il s'agit d'un projet collaboratif consistant à poser des pixels, afin de réaliser des illustrations par exemple. Durant ce projet, j'ai consolidé mes connaissances sur JavaScript et j'ai appris à gérer une API afin de mettre à jour le tableau de pixels dès qu'un étudiant le modifiait. C'est jusque-là mon projet préféré de mes études en informatique puisque nous avons pu lier le côté amusant et créatif de la pixel war avec le web.
